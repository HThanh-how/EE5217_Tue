8.4. Đóng gói Artifact PDF Report

Quy trình Release tự động

1
Compile XeLaTeX

Biên dịch main.tex → EE5217_BaoCao.pdf

2
Upload Artifact

Lưu PDF vào kho GitHub Artifacts để truy xuất sau

3
Generate Changelog

Tự động tổng hợp lịch sử commit thành Release Notes

4
Create GitHub Release

Đóng gói phiên bản chính thức kèm file PDF tải về

build.yml → release job 
release:
  if: github.event_name == 'push'
  needs: build-pdf
  steps:
    - name: Generate changelog
      run: |
        git log --pretty=format:"* %s (%h)"
        > release_notes.md

    - name: Create Release
      uses: softprops/action-gh-release@v2
      with:
        tag_name: build-${{ run_number }}
        name: Báo Cáo EE5217
        body_path: release_notes.md
        files: EE5217_BaoCao.pdf
