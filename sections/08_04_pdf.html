8.4. Đóng gói Artifact PDF Report

  • Đường ống CI/CD còn gánh thêm nhiềm vụ tự động Compile file report.pdf cuối cùng.
  • File PDF được gắn thẻ và đính kèm trực tiếp vào mục GitHub Artifacts để tải về.
  • Cơ chế GitHub Releases tự động được tạo ra giúp lưu trữ các phiên bản báo cáo theo mốc thời gian vĩnh viễn và chuyên nghiệp như những phiên bản phần mềm thương mại.
