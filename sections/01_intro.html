1. Đặt Vấn Đề

  • Hệ thống Boilerplate này giúp quản lý 2 định dạng: Slide Trình Chiếu và Bản in PDF.
  • Bản in PDF được tự động sinh (Compile) qua GitHub Actions với LaTeX.
  • Trình chiếu slide sử dụng HTML thuần, có thể truy cập bằng GitHub Pages bất cứ đâu.
  • Giúp loại bỏ sự rườm rà khi phải sử dụng Microsoft Word và PowerPoint cùng lúc.
