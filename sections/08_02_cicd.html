8.2. Đường ống CI/CD với GitHub Actions

Cơ chế Kích hoạt tự động

  • Ngay khi có sự kiện git push lên nhánh Main, nền tảng GitHub sẽ cấp phát một máy chủ (Runner) độc lập trên Cloud.
  • Máy ảo này tự động clone mã nguồn, chạy Container chứa môi trường biên dịch TeX hoặc Nodejs để render Web.
  • Từ đó không cần bất cứ thành viên nào thao tác bằng tay trên máy tính cá nhân. Quá trình mất chưa tới 40 giây.
