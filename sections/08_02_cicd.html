Thao tác tay
build.yml 
name: Build LaTeX PDF
on: [push, pull_request]

jobs:
  build-pdf:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Compile LaTeX
        uses: xu-cheng/latex-action@v3
        with:
          root_file: main.tex
          latexmk_use_xelatex: true

  release:
    needs: build-pdf
    steps:
      - uses: softprops/action-gh-release@v2
        with:
          tag_name: build-${{ run_number }}
          files: EE5217_BaoCao.pdf

Hoàn toàn tự động — Không cần bất kỳ thành viên nào cài đặt LaTeX trên máy cá nhân.
