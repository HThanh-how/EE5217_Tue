8.3. Triển khai Trình chiếu lên GitHub Pages

Web Server Serverless

  • Biến Repository thành một trang Web mã nguồn mở tĩnh thông qua GitHub Pages an toàn, bảo mật.
  • Báo cáo dạng thẻ HTML thuần túy (Reveal.js) dễ dàng tích hợp Script Python chạy Audio ẩn tàng hình, hoán chuyển mạch lạc như PowerPoint.
  • Khán giả hoặc giáo viên quét mã QR là có thể theo dõi trực tiếp Slide trên điện thoại Real-time!
